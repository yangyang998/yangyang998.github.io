3.添加java来实现LocaleResolver方法，重写resolveLocale setLocale方法

public class MyI18n implements LocaleResolver {
    @Override
    public Locale resolveLocale(HttpServletRequest request) {
    	/**
    	 * 通过lang获取请求头 zh_CN或en_US
    	 * split("_")通过"_"分割请求头
    	 */
        String lang = request.getParameter("lang");
        Locale locale = Locale.getDefault();        //设置默认
        if (!StringUtils.isEmpty(lang)){
            String[] split = lang.split("_");
            locale = new Locale(split[0], split[1]);
        }
        return locale;
    }

    @Override
    public void setLocale(HttpServletRequest request, HttpServletResponse response, Locale locale) {

    }
}
  • 并在自定义配置中添加把自定义的配置注入Bean

    public LocaleResolver localeResolver(){
        return new MyI18n();
    }

7.登录 重定向 与拦截器

1.添加一个 Controller java文件

@Controller
public class LoginController {

    @PostMapping("/login/showpage")
    //没有从数据库 只要判断登录名不为空，密码为1就可以提交登录
    public String Login(@RequestParam("loginName") String login,
                        @RequestParam("password") String password,
                        Map<String,Object> map,
                        HttpSession session){

        if (!StringUtils.isEmpty(login)&&"1".equals(password)){
            //重定向，防止表单重复提交  通过session防止通过地址直接访问
            session.setAttribute("loginName",login);
            return "redirect:/management.html";
        }

        else{
            map.put("msg","用户名或密码错误！");
            return "login";
        }
    }
}
  • 1.return "redirect:/management.html";中的redirect:是重定向到根目录下的management.html
  • 2.在自定义配置config下面添加映射到实体文件下
registry.addViewController("/management.html").setViewName("info");
  • 3.@RequestParam("loginName")获取form表单上面的Name值

2.自定义一个拦截器实现HandlerInterceptor方法，重写preHandle

@Controller
public class MyHandlerInterceptor implements HandlerInterceptor {
    @Override
    public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) throws Exception {
        Object login = request.getSession().getAttribute("loginName");
        if (login==null){
            request.setAttribute("msg","没有权限，重新登录！");
            request.getRequestDispatcher("index.html").forward(request,response);
            return false;
        }
        else{
            return true;
        }
    }
}
  • 1.getAttribute("loginName");获取表单的Name值
  • 2.重写WebMvcConfigurer下面的addInterceptors方法

            @Override
            public void addInterceptors(InterceptorRegistry registry) {
                registry.addInterceptor(new MyHandlerInterceptor()).addPathPatterns("/**")
                        .excludePathPatterns("/","/index.html","/login/showpage/**","/info-style/**","/login-style/**");
            }
  • 3.通过addPathPatterns()拦截 ,excludePathPatterns()排除拦截

@Repository
public class EmployeeDao {

    private static Map<Integer,Employee> employeeMap =null;

    @Autowired
    private DepatmentDao depatmentDao;

    static {
        //创建一个员工表
        employeeMap =new HashMap<Integer, Employee>();

        //给员工添加数据
        employeeMap.put(1001,new Employee(1001,"张三","6160@126.com",1,new Department(101,"总经办")));
        employeeMap.put(1002,new Employee(1002,"李四","6160@126.com",0,new Department(102,"工程部")));
        employeeMap.put(1003,new Employee(1003,"王五","6160@126.com",0,new Department(103,"集成部")));
        employeeMap.put(1004,new Employee(1004,"赵六","6160@126.com",1,new Department(104,"开发部")));
        employeeMap.put(1005,new Employee(1005,"钱七","6160@126.com",1,new Department(105,"后勤部")));
    }
    //设置默认id值，
    private static Integer initId =1006;

    //增加一个员工
    private void add(Employee employee){
        if (employee.getId()==null){
            employee.setId(initId++);
        }
        employee.setDepartment(depatmentDao.getDepartmentById(employee.getDepartment().getId()));
        employeeMap.put(employee.getId(),employee);
    }
    //查询全部员工信息
    public Collection<Employee> getAll(){
        return employeeMap.values();
    }
    //通过id查询员工
    public Employee getEmploeeById(Integer id){
        return employeeMap.get(id);
    }
    //删除员工通过id
    public void delete(Integer id){
        employeeMap.remove(id);
    }

}

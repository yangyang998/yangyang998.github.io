hugo搭建博客笔记

Posted by YangYang on Wednesday, March 11, 2020

搭建个人博客

1.准备工作

  • 一.下载git工具：https://git-scm.com/downloads

  • 二.下载hugo静态网站生成器：https://github.com/gohugoio/hugo/releases

  • 三.需要github账户把网站部署到远端，方便通过Inter网访问

  • 四.下载模板通过git下载，模板网站：http://themes.gohugo.io/

2.开始

  • 一. 配置hugo，git的环境变量，方便通过cmd命令调用工具

  • 二. 通过hugo创建我的第一个项目,命名为myblog，参考官方文档

    hugo new site myblog
    • 创建完过后会生成以下文件
    ├─archetypes
├─content
├─data
├─layouts
├─static
└─themes

  • 三.通过git clone下载模板到themes文件目录下

    git clone https://github.com/zhaohuabing/hugo-theme-cleanwhite.git

  • 四.在本地运行hugo服务 需要更改配置config.toml来显示界面

    hugo server -t [模板名称] -D

    运行:http://localhost:1313

  • 五.通过hugo打包发布到github

    hugo -t=[模板名称] -b="[仓库地址]" -D

    添加到github

    git init
git add README.md
git commit -m "first commit"
git remote add origin https://[github地址]/[仓库地址]
git push -u origin master

    输入用户名，密码后登录github上刷新。

更新博客

  • 第一步：查看当前的git仓库状态

    git status

  • 第二步：更新全部

    git add *

  • 第三步：接着输入git commit -m “更新说明”

    git commit -m “更新说明”

  • 第四步：先git pull,拉取当前分支最新代码（更新本地代码）

    git pull

  • 第五步：push到远程master分支上（更新GitHub上的代码）

    git push origin master

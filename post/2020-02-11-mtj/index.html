mysql tomcat java环境配置

Posted by YangYang on Tuesday, March 10, 2020

1.搭建jdk8环境

  • 部署jdk8解压缩jdk8压缩包到自定义【mysoft】文件夹

  • 配置java 的环境变量在 计算机===>属性===>高级属性设置===>环境变量：

    • JAVA_HOME 变量，值为jdk8 的安装路径;
    • Path 变量，值为%JAVA_HOME%\bin;
    • CLASSPATH 变量，值为:%JAVA_HOME%\lib;%JAVA_HOME%\jre\lib;

  • 完成！

2.搭建Tomcat8环境

  • 部署 Tomcat8解压缩Tomcat8到自定义【mysoft】文件夹

  • 配置java 的环境变量在 计算机===>属性===>高级属性设置===>环境变量：

    • CATALINA_HOME变量，值为Tomcat8的安装路径;
    • Path 变量，值为%CATALINA_HOME%\bin;

  • cmd 运行catalina -version查看依赖环境,startup运行Tomcat

  • 解决控制台乱码情况安装目录===>conf===>logging.properties文件

    • 更改： java.util.logging.ConsoleHandler.encoding = GBK

  • 完成！

  • Linux下安装Tomcat

    官网下载地址：
    apache-tomcat-8.5.51.tar.gz

  • 解压下载文件重命名

  • 如果是Windows上面下载的需要传到Linux上需要通过psftp.exe工具找到该文件cmd进入路径

    [psftp文件路径]>psftp
psftp> open [ip地址]
put [文件路径]\[文件名称]

  • 解压Tomcat文件，并重命名

    tar -zxvf apache-tomcat-8.5.51.tar.gz
mv apache-tomcat-8.5.51 tomcat8
mv tomcat8 Application/tomcat8

  • vim编辑/etc/profile文件

    export CATALINA_HOME=[文件路径]/[安装目录]

  • 使配置生效

    source /etc/profile

  • 启动配置

    cd [安装路径]/bin
./startup.sh

3.搭建mysql环境

  • 部署 mysql 解压缩mysql 到自定义【mysoft】文件夹

  • 在解压后的bin 目录下添加my.ini文件：

    [client]

port=3306

[mysql]

default-character-set=utf8mb4

#no-beep

[mysqld]

port=3306

# mysql根目录

basedir=""

# 放所有数据库的data目录

datadir=

# 默认存储引擎innoDB

default-storage-engine=INNODB

# 即日志跟踪，1为开启，0为关闭

general-log=0

general_log_file="mysql日志.log"

#错误信息文件设置，会将错误信息放在data/mysql.err文件下

#log-error=mysql.err

# 1  表名存储在磁盘是小写的，但是比较的时候是不区分大小写

# 0  表名存储为给定的大小和比较是区分大小写的 

# 2  表名存储为给定的大小写但是比较的时候是小写的

lower_case_table_names=1

# 最大连接数

max_connections=151

# 打开表的最大缓存数

table_open_cache=2000

# tmp_table_size 控制内存临时表的最大值

tmp_table_size=16M

# 最大缓存线程数量

thread_cache_size=10

  • MYSQL_HOME 变量,值为MYSQL的安装路径; Path变量,值为%MYSQL_HOME%\bin

  • 初始化数据库：

    mysqld --initialize --console

  • 安装mysql服务:

    mysqld -install

  • 创建data目录命令：

    mysqld  --initialize-insecure

  • 启动mysql服务:

    net start MySQL

  • 初始密码为空添加密码命令:

    set password=password('你的密码');
alter user 'root'@'localhost' PASSWORD EXPIRE NEVER;
flush privileges;

  • 完成！
